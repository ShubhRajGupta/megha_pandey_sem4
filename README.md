# Tetris Game Using Python OOP and Pygame

## Project Overview
This project is a fully functional Tetris game developed using Python and the Pygame library. The project demonstrates the implementation of Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, abstraction, and modular design.

---

## Objective
The objective of this project is to design and implement an interactive Tetris game using Python and OOP principles.

### Problem Statement
Managing game logic using procedural programming becomes difficult as the project size increases.

### Solution
This project solves the problem by organizing gameplay features such as movement, collision detection, scoring, and line clearing into separate classes and methods.

### Why OOP is Used
OOP is used to improve code organization, modularity, reusability, scalability, and maintainability.
